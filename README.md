# 双语文档

## 介绍

本项目旨在发展和推广中英双语对照形式的文档。通过提供原文和译文对照的形式，帮助用户更易理解和使用文档内容。

## 双语对照形式文档的优势

更易理解：将原文和译文相互对照参考，便于理解文中的专业术语和内容。
